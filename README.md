# TopDown Shooter

## Description
Top-down shooter with wave system, enemies, bosses, weapons system,  and player combat.

## Features
- Wave system
- Enemy AI (melee / ranged)
- Player shooting system
- UI system
- Game states (Menu / Gameplay / GameOver)

## Tech
- Unity (URP)
- C#
- ScriptableObjects
- Event-driven architecture

## Architecture
- SRP (Single Responsibility)
- Decoupled systems (UI separate from gameplay)
- Event-based communication

## How to run
Open project in Unity 2021.3+
